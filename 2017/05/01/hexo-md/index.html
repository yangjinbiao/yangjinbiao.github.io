hexo使用方法

基础目录和配置

  • 使用的是 hexo3
  • github 的仓库地址 git博客.
  • 博客目录在 ~/blog/content
  • nodejs 安装路径 ~/blog/node-v6.10.2-linux-x64

  • 有时候更新你不成功 需要更改nodejs的仓库地址为淘宝的地址 .

  • 主题只是使用了next 

    >   deepin下的安装方式 使用的是 从官网下载压缩包解压的方法,需要export路径，要不然找不到命令

相关配置

  • 作者，标题，简介都在 站点配置文件(_config.yml)里面
  • 博客主题样式可以有多个
  • 配置选项后面必须有个空格
